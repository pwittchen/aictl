# aictl — Project Instructions

This is the aictl repository, an AI agent CLI tool written in Rust.

- The project is built with `cargo build` and tested with `cargo test`.
- Configuration lives in `~/.aictl/config` as key=value pairs.
- Source code is in the `src/` directory. The entry point is `src/main.rs`.
- When discussing this project, be specific about file names and module boundaries.
- Issues and planned work are tracked in `ISSUES.md` at the repository root.
